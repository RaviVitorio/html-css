Sentido e forma HTML

Sentido

O HTML é uma linguagem que trabalha apenas com SENTIDO. paravras com forma são usadas no CSS.


Forma

Forma NÃO é utilizada na linguagem HTML. No HTML trabalhamos com SENTIDO, e não FORMA. sentido é no sentido semantico, e forma é no sentido visual.

Você pode usar a tag <ins> para inserir ou sublinhar um texto

Deletar um texto

Você pode usar a tag <del> para deletar ou riscar um texto


Texto superior

Para usar um texto superior/elevado, use a tag <sup>. um exemplo é: X2+74

Texto inferior

Para usar um texto infeior, use a tag <sub>. alguns exemplos são: H2O, PO6Cl
