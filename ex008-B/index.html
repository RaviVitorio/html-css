Exibição facilitada de codigos

Podemos facilitar a leitura de codigos utilizando a tag <code> para fazer com que o texto fique mono-espaçado. Por exemplo:

num = int(input('Digite um número')) if num%20: print(f'0 número (num) é par') else: print(f'o número {num} é impar') print('FIM DO PROGRAMA')


E podemos facilitar mais ainda utilizando a tag <pre> antes da tag <code>. Por exemplo:

        
    num = int(input('Digite um número')) if num%20:
        print(f'0 número (num) é par') else: print(f'o número {num} é
        impar') print('FIM DO PROGRAMA')
